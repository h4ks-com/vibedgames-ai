🎮 Base-4 Block Encoding Demo

Reading 4-color cells in groups — does bigger blocks = better compression?

The idea: Instead of reading one 4-color cell at a time (4 possible values → tiny dictionary), read multiple cells as one symbol. 2 cells = 4² = 16 words, 3 cells = 64 words, 4 cells = 256 words. The catch: Total information never changes — you're just slicing the same bits differently.
