§7 — Gödel's First Incompleteness Theorem

The Unprovable Tacos

We now arrive at the heart of the matter. Gödel showed that any sufficiently powerful formal system contains statements that are true but unprovable within the system. We demonstrate this entirely within ℒTℵP.

THE GÖDEL SENTENCE FOR ℒTℵP
GTℵP : ¬(∃ proof : ⊢ ⟨GTℵP⟩)
"This sentence about tacos al pastor cannot be proven within the Tacos al Pastor formal system."
In plain language: GTℵP says of itself that no valid proof in 𝒯TℵP derives it.

Proof Sketch — The Diagonalization Lemma

1.
We need a self-referential sentence. The trick is the diagonal lemma: For any formula φ(x) with one free variable,
there exists a sentence ψ such that 𝒯TℵP ⊢ ψ ↔ φ(⌜ψ⌝) In ℒTℵP: there exists a taco-sentence that asserts "the proof-number of this very sentence satisfies property φ."
2.
Define the provability predicate for tacos: ProvTℵP(n) ≡ "n is the Gödel number of a sentence provable in 𝒯TℵP"
≡ "n encodes a provable taco-statement" This predicate is itself expressible in ℒTℵP (because proofs are finite, verifiable objects).
3.
Apply the diagonal lemma with φ(x) = ¬ProvTℵP(x): ∃ GTℵP : 𝒯TℵP ⊢ GTℵP ↔ ¬ProvTℵP(⌜GTℵP⌝) This GTℵP is our Gödel sentence for tacos al pastor.
4.
Suppose 𝒯TℵP ⊢ GTℵP. Then there exists a proof with Gödel number n = ⌜GTℵP⌝, so ProvTℵP(⌜GTℵP⌝) holds. But GTℵP asserts ¬ProvTℵP(⌜GTℵP⌝). Contradiction. ∴ 𝒯TℵP ⊬ GTℵP
5.
Therefore ¬ProvTℵP(⌜GTℵP⌝) is true (since GTℵP is indeed not provable). By the equivalence in step 3, GTℵP is true. 𝒯TℵP ⊬ GTℵP ∧ 𝒯TℵP ⊬ ¬GTℵP GTℵP is true but undecidable in 𝒯TℵP. QED ∎
