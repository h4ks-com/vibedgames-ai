Mixture of tiny experts

grow a pool of specialists, one boolean function at a time.

weight bits
9
search space
512
target family
all 16 two-input boolean functions
method
exhaustive per task, checkpointed growth

Start with an empty pool. For each of the 16 two-input boolean functions, search the 512-model space for a solver. If found, add it to the pool and (in the C version) checkpoint atomically. Resumable from any interruption. Runtime cost per decision: one dispatch by task_id plus one LUT read.
